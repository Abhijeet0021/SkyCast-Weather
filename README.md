<h1>🌤️ SkyCast – Weather Forecast Web App </h1>

<p>SkyCast is a responsive weather forecasting web application built using HTML, CSS, and Vanilla JavaScript.<br>It integrates with a Weather REST API to provide real-time weather updates, a 5-day forecast, geolocation-based detection, temperature unit conversion (°C/°F), and favorite city management using localStorage.</p>

  <h1>🚀 Features</h1><br>
  
🌍 Search weather by city name<br>
📍 Detect weather using current location (Geolocation API)<br>
🌡️ Toggle temperature units (°C / °F)<br>
📅 5-Day forecast display<br>
⭐ Save favorite cities (stored in localStorage)<br>
🎨 Glassmorphism UI with smooth animations<br>
⏳ Loading and error state handling<br>
📱 Fully responsive design<br>

<h1>🛠️ Tech Stack</h1>
 <li> HTML5 – Structure</li>
 <li> CSS3 – Styling & Animations</li>
 <li> JavaScript (ES6+) – Application Logic</li>
 <li> Fetch API – API Integration</li>
 <li> LocalStorage – Persistent Favorites</li>
 <li> Geolocation API – Location Detection</li>

<h1>🔄 Application Flow</h1>
1 Application loads with a default city.</n>
2 User searches a city or uses current location.
3 App sends request to Weather API using fetch().
4 API returns JSON data.
5 JavaScript processes data and updates the DOM dynamically.
6 User can toggle units or save city without page reload.
